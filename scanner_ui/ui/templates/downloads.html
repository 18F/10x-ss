API and download access

The Federal Website Scanner offers API access and downloadable data files for all scan types and data layers.

File scanner

The file scanner checks for the presence or absence of specific files at specific locations including /code.json, /data.json, /data, /developer, /digitalstrategy, /open, /privacy, /robots.txt, and /sitemap.xml.

  • Download report for all domains: JSON | API | CSV
  • Customize report for an individual domain (e.g. 18f.gov): JSON | API

Sitemap scanner

The sitemap scanner checks for the presence or absence of a /sitemap.xml file. If the file is present, further analysis is provided.

  • Download report for all domains: JSON | API
  • Customize report for an individual domain: JSON | API

Pagedata Scanner

  • For all domains - JSON - API
  • For an individual domain (e.g. 18f.gov) - JSON - API

Privacy Scanner

  • For all domains - JSON - API
  • For an individual domain (e.g. 18f.gov) - JSON - API

USWDS Scanner

  • For all domains - JSON - API
  • For an individual domain (e.g. 18f.gov) - JSON - API

All Scans Combined

  • For all domains - JSON - API
  • For an individual domain (e.g. 18f.gov) - JSON - API
