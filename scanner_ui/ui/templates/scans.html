List of Scans

This page lists each of the active scans and provides more information about them.

200 Scanner

A scan to check each domain for the presence or absence of a file at a specific location, specifically by analyzing the server response code at that location.

Learn more about the scan here.

USWDS Scanner

A scan to check each domain for the presence or absence of a US Web Design System (USWDS) component (i.e. font, style sheet) and the version in use of USWDS.

Learn more about the scan here.

Pagedata Scanner

A scan to check each domain for the presence or absence of certain canonical files at specific locations and to analyze the characteristics of the pages that it finds.

Privacy Page Scanner

A scan to check each domain for the presence or absence of a privacy page and to analyze the characteristics of the pages that it finds.

Sitemap Scanner

A scan to check each domain for the presence or absence of a sitemap.xml file and to analyze the characteristics of the files that it finds.
