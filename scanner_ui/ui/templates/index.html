How to Use this Site

There's four main sections to this website:

  • Use Cases: Short descriptions of how different users could use the different presentation layers to interact with scan data.
  • Presentation Layers: An index of different dashboards and mashups that provide an interface to the scan.
  • Data Links: Direct download and API links for the scan data.
  • About Each Scan: Information about each scan, how it works, and what it does.
