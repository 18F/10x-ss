List of Potential Use Cases

This page lists different scenarios and workflows for using the data and presentation layers made possible by these scans.

Code.gov Team

Task: Find list of active code.json files

Process:

  1. Navigate to this presentation layer

Task: Analyze the characteristics of the code.json files

Process:

  1. Navigate to the code.json dashboard
  2. Filter for relevant agency, branch, or organization
  3. Note the json items, file size, and code.gov measurement type fields to the far right of the chart

Data.gov Team

Task: Find list of active data.json files

Process:

  1. Navigate to this presentation layer

Task: Analyze the characteristics of the data.json files

Process:

  1. Navigate to the data.json dashboard
  2. Filter for relevant agency, branch, or organization
  3. Note the json items and opendata conformity fields to the far right of the chart

Search.gov Team

Task: Analyze the characteristics of an agency's sitemap.xml files

Process:

  1. Navigate to the sitemap.xml dashboard
  2. Filter for relevant agency, branch, or organization
  3. Note the URL count and locations in the robots.txt fields to the far right of the chart

Task: Analyze the characteristics of an agency's robots.txt files

Process:

  1. Navigate to the robots.txt dashboard
  2. Filter for relevant agency, branch, or organization
  3. Note the content type and file size fields to the far right of the chart

USWDS Team

Task: Find list of domains that are implementing the USWDS

Process:

  1. Navigate to the USWDS dashboard
  2. Choose a score such as `50` to input into the Score Filter so as to filter the results for domains that have implemented USWDS past a certain degree.

https://site-scanning.app.cloud.gov/privacy/

Agency Privacy Officer

Task: Analyze the characteristics of an agency's privacy pages

Process:

  1. Navigate to the Privacy Page dashboard
  2. Filter for relevant agency, branch, or organization
  3. Note the emails and H1, H2, H3 Headers columns to the far right of the chart
