Use cases

The Federal Website Scanner was designed to meet the needs of an initial set of federal partners. We worked with partners to understand the types of scanning tasks they wanted to perform and how they wanted to access the results. If you have use cases to contribute please let us know! E-mail site-scanning@gsa.gov or work with us on GitHub.

Code.gov team

Challenge: The Code.gov team needed on-demand access to a list of active code.json files and a method for analyzing those results.

Solution: Use the File Scanner to search for the presence of code.json files across all government websites or a subset of sites. Results can be filtered by branch, agency, or domain. Results include characteristics of the code.json files. A code.json dashboard was developed as a proof-of-concept.

  1. Navigate to this presentation layer

Task: Analyze the characteristics of the code.json files

Process:

  1. Navigate to the code.json dashboard
  2. Filter for relevant agency, branch, or organization
  3. Note the json items, file size, and code.gov measurement type fields to the far right of the chart

Search.gov Team

Task: Analyze the characteristics of an agency's sitemap.xml files

Process:

  1. Navigate to the sitemap.xml dashboard
  2. Filter for relevant agency, branch, or organization
  3. Note the URL count and locations in the robots.txt fields to the far right of the chart

Task: Analyze the characteristics of an agency's robots.txt files

Process:

  1. Navigate to the robots.txt dashboard
  2. Filter for relevant agency, branch, or organization
  3. Note the content type and file size fields to the far right of the chart

USWDS Team

Task: Find list of domains that have implemented parts of the USWDS

Process:

  1. Navigate to the USWDS dashboard
  2. Choose an analysis count such as `50` to input into the Analysis Filter so as to filter the results for domains that have implemented USWDS past a certain degree.

Agency Privacy Officer

Task: Analyze the characteristics of an agency's privacy pages

Process:

  1. Navigate to the Privacy Page dashboard
  2. Filter for relevant agency, branch, or organization
  3. Note the emails and H1, H2, H3 Headers columns to the far right of the chart
